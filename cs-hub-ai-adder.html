⚙️ One-time setup required

Open this file in a text editor and set your values at the top of the script:
OPENROUTER_API_KEY — your free API key from openrouter.ai → Sign in → Keys → Create Key
GOOGLE_SCRIPT_URL — your Apps Script Web App URL (for writing to the sheet, see Step 3 below)

Add a Resource with AI

Paste any URL — a YouTube video, Google Slides, BBC Bitesize page, worksheet, anything. The AI will read the page, decide which year group, topic and lesson it belongs to, suggest a title and type, then let you confirm before adding it to your Google Sheet.
